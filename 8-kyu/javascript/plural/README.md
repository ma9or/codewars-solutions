Она должна принимать число и возвращать true, если с этим числом следует использовать множественное число, или false, если нет. Это было бы полезно при выводе строки типа </font></font><code>5 minutes</code><font dir="auto" style="vertical-align: inherit;"><font dir="auto" style="vertical-align: inherit;">, </font></font><code>14 apples</code><font dir="auto" style="vertical-align: inherit;"><font dir="auto" style="vertical-align: inherit;">, или </font></font><code>1 sun</code><font dir="auto" style="vertical-align: inherit;"><font dir="auto" style="vertical-align: inherit;">.</font></font></p><blockquote><p><font dir="auto" style="vertical-align: inherit;"><font dir="auto" style="vertical-align: inherit;">В этой ката вам нужно беспокоиться только о правилах английской грамматики: всё, что не является единственным числом (один из чего-либо), является множественным числом (не один из чего-либо).</font></font></p></blockquote><p><font dir="auto" style="vertical-align: inherit;"><font dir="auto" style="vertical-align: inherit;">Все значения будут положительными целыми числами, числами с плавающей запятой или нулем.</font></font></p>
